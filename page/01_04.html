Various events can be created in Scene Builder.
* Essentially all events are compatible with MV.

Show Picture Event

The event lengths of Show Picture Events can be set, with the lowest setting at one frame. These are collected as MV picture related events, and can be moved or have their hues change over time.

Picture

Settings for the image displayed.

Number
The picture's number. The higher the number, the higher it is placed when stacking. When two picture numbers are the same, the one in the lower row will take priority.
Image
Select the picture to display.

Start Position

This setting is for the initial display position of the picture.

Origin
Select the center location for when a picture is transformed or rotated.
X: Y: Width: Height:
Set the picture's position and size.
Rotation Speed
Set the rate for the picture's rotation.

Combination

Set picture's start time combination.

Opacity
Configure the opacity of the picture.
Blend Mode
Specify the combination method for overlapping pictures.

End Position

Set the position of the picture's display at its ending time.

X: Y: Width: Height:
Set the picture's position and size.

End Blend

Set the combination method for the picture's display at its ending time.

Opacity
Configure the opacity of the picture.

Color Tone

Set the color tone of the picture's display at its ending time.

Show Text Event

The length of a Show Text Event cannot be set. Also, you cannot display more than one text item during the same frame. In the case that more than one is set to the same frame, the lower one will take priority.

Message

Face
Set the face image to be displayed within the text window.
Text
Input the text to be displayed during the event. More than four lines of text will be continued in a subsequent event.
Background
Set the window's background.
Window Position
Set the window's position.

Show Scrolling Text Event

The length of a Show Scrolling Text Event cannot be set. Also, you cannot display more than one text item during the same frame. In the event that more than one is set to the same frame, the text on the lower row will take priority. All control characters can be used.

Scrolling Text

Text
Input the text to be scrolled over display.
Speed
Set the scroll speed.
No Fast Forwarding
If this is checked, keys cannot be used to fast forward through the scrolling text.

Fade Screen Event

Set up a Fade Event. The length of a Fade Screen Event is fixed at 24 frames. It is necessary to perform a Fade Out before performing a Fade In. Fade Type.

Fade Type

Fade Type
Set if the selected event is a Fade Out or a Fade In.

Screen Event

Add an event that has an effect on a picture. Select from the two types: Flash or Shake.

Flash Color

　

Checking this will cause a flash effect to occur. The tint and strength can be set.

Shake

　

Checking this will cause a shake event to occur. The strength and speed can be set.

Audio Event

Set up an Audio Event. The length of an Audio Event cannot be set. Also, multiple Audio Events cannot occur at the same time.

Audio

Audio Type
Select the type of audio.
Audio
Select the sound source file.

Fadeout Audio Event

Add an Audio Fade-Out Effect. The fade-out time is set by the length of the frame.

Audio Type

Audio Type
Specify the type of the fade-out audio.

Marker

Set a marker on the timeline. Annotations can be added by clicking on a marker.

Preset

A Preset is scene data that has been put together beforehand. When a Preset Scene uses images or audio materials, the files will be copied and overwritten into the project folder.
